# my-keyboard
I'm making a mechanical keyboard, from scratch.

## Design
The theme of this keyboard is a minamalistc gaming keyboard that can fit in a clean room.

## What did I make exactly?
In this project, I made 60% mechanical keyboard from scratch. This means I designed the PCB, The Case and I am going to assemble this together, including soldering.

## What have I learned?
Upon finishing this project I will have learned:
-Basic Kicad Skills
-Further Concrete Onshape Skills
-Using software with my Keyboard
-Soldering!
-Touch Typing? (maybe)

## How I designed this
I used Kicad for the custom PCB, and onshape for the case.

## Why I made this
I wanted to make a keyboard from scratch to learn new skills, and have an awesome keyboard sitting in my room that looks super cool
